# 2. 📖 Descrição do Projeto

Este projeto consiste em um sistema simples de gerenciamento de produtos desenvolvido em Java, com foco no estudo e aplicação de tratamento de exceções.

O objetivo principal é simular operações básicas de um sistema real (como cadastro, busca e remoção), garantindo validações e tratamento adequado de erros, evitando falhas inesperadas durante a execução.

---

# 3. ⚙️ Funcionalidades

* Cadastro de produtos
* Listagem de produtos
* Busca de produto por nome
* Remoção de produto
* Tratamento de exceções personalizadas:

  * Produto não encontrado
  * Valor inválido

---

# 4. 🛠 Tecnologias Utilizadas

* Java 25
